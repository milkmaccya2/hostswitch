# はじめに

**HostSwitch** へようこそ！

HostSwitchは、開発者のために作られたシンプルなhostsファイル切り替えツールです。
複数の環境（ローカル、ステージング、本番など）を行き来するWeb開発者や、サーバー管理者の作業を効率化します。

## 主な特徴

- **⚡ 高速な切り替え**: コマンド一発でhostsファイルを切り替えられます。
- **🛡 安全設計**: 現在のhostsを自動バックアップしてから切り替えるため、事故を防げます。
- **📝 対話的インターフェース**: コマンドを覚えなくても、矢印キーで直感的に操作できます。
- **📂 プロファイル管理**: 環境ごとに設定を「プロファイル」として保存・管理できます。
